When we make a worflow, and everthings work scquencely, and also involve AI we can use LangGraph.
    npm i langchain @langchain/langgraph @langchain/google-genai @langchain/core.

//Imports:
    - {StateGraph} - creating a state graph
    - {type GraphNode} - creating multiple nodes in the graph
    - {StateSchema} - defining formats of the data that will be passed between nodes.
    - z - for type safety and validation of the data in runtime.

//Creating a state graph:
    - In state, we define the data that will be passed between nodes, and also the format of the data.

//Agent:
    - When user want data in perfect format, we can use agent to do that.

    - import {createAgent, providerStrategy} from "langchain";  
    - createAgent - to create an agent
    - providerStrategy - to define how the agent will interact with the AI provider (e.g., Google GenAI).
    - toolsStrategy - As same as providerStrategy, but when user use another AI tools expect google, then user can use it.
